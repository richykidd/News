# News

这个一款使用Swift3.0编写的资讯新闻类app，项目首页使用使用网易首页选项卡方式切换控制器

###新闻客户端, 待完善```
图库详情使用传统新闻客户端的图片浏览器，可以隐藏/显示UI，缩放图片，保存图片，多种手势操作。
集成推送、第三方分享、第三方登录等SDK，项目代码注释清晰!
